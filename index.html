Covid Calculator Canada /

A web app created with Vue.js and Python to mathematically predict the threat level posed to the user by COVID-19.

Uses a variety of different factors to calculate risks, including pre-existing conditions, current symptoms, and the statistics from the users current geographical location within Canada.

  • I successfully designed and implemented the algorithm to determine the likelihood that the user's current symptoms are caused by COVID-19.
  • Collaboratively created the user interface, ensuring it is easy and straightforward to use.

Created for Biomedical Computing (CMPT 340) - April 2020

EatSafe Surrey /

An android app that allow users to see the latest health inspections at any restaurant in Surrey, BC.

As part of my team, I...

  • Implemented multiple classes for handling data and displaying it in a user-friendly format.
  • Created the ability for users to favourite restaurants so they can find them from the list quickly.
Map Screen - A scrollable and pannable map that lets the user visually select their favourite restaurants
Restaurant Info Screen - User can tap on an inspection from the list to open the inspection info screen
Inspection Info Screen - Shows hazard level and individual health code violations for each health inspection
